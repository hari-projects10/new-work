Clickart – AI-Powered eCommerce Website

Project Description

Clickart is a fully functional and responsive eCommerce platform built using PHP and modern front-end technologies. Designed to deliver a smooth shopping experience, it offers all the essential features of a typical online store, from product browsing to secure checkout.

Key Features

  • Product Catalog: Browse a wide range of items with detailed listings and images.
  • Secure Checkout: Supports complete checkout flow with order summary and confirmation.
  • Clickart AI Integration: Built with Gemini's private AI API to provide smart product suggestions and search assistance.
  • AI Chatbot: Helps users find products quickly and interactively through conversational search.
  • Responsive Design: Optimized for all devices to ensure seamless usability.

This project combines eCommerce functionality with intelligent AI features, offering users a modern, efficient, and interactive shopping experience.
